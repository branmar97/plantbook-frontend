# Plantbook Frontend

A social space for gardeners and plant enthusiasts. Find, favorite and view information on over 35,000 plants. 

## Usage

Signup or login to use the application.

## Built Using

- ReactJS
- Redux
- Redux Thunk
- Ruby on Rails
- HTML/CSS
